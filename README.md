
# 사용자가 정의하는 단어사전 HelloWord!

## 개발환경
  1. 개발언어
    · JAVA
  2. 개발 IDE
    · Eclipse기반의 STS
  3. 코드 형상관리
    · Git
  4. 코드 빌드
    · Maven
  5. Database
    · MySQL

## 프레임워크 
  1. 웹 어플리케이션 
    · SpringMVC
  2. Database
    · MyBatis
  3. WAS
    · Tomcat
 
## 프로젝트 소개
  1. 사용자가 직접 정의할 수 있는 신조어 사전 웹사이트 
  2. 나머지 정의 또한 신뢰도 순으로 볼 수 있음
  3. 사용자가 신조어를 자유롭게 정의하고 추천을 많이 받은 단어가 상위에 노출되어 사용자에게 신뢰도 높은 단어로 노출

## 앞으로 개발 계획
  1. 기능 > REST API로 변경 
  2. Eclipse > IntelliJ
  3. SpringMVC > Spring Boot
